Using the overlay
  1. Install Tampermonkey (Chrome, Edge or Opera), or Violentmonkey (Firefox). Android users can use Kiwi Browser and install the script as usual.
  2. Go to https://greasyfork.org/en/scripts/442576-r-steinsplace-overlay and install the script by pressing "Install".
  3. Refresh the r/place page.
